Glow Me is a **modern and responsive cosmetics shopping app** that allows users to browse, favorite, and purchase makeup products with ease. It combines smooth animations, a modern UI, and secure payment integration to deliver a seamless shopping experience.

---

## üåü Features

- **User Authentication**
  - Sign up & login using **Firebase Authentication**  
  - Email & password confirmation  
  - Password reset functionality  

- **Shopping & Product Management**
  - Browse products from **[Makeup API](https://makeup-api.herokuapp.com/)**  
  - View products randomly or by category/type  
  - Add items to **Cart** or **Wishlist** using **Provider**  
  - Store data locally using **SharedPreferences** (cached items)  

- **User Interface**
  - Fully **responsive UI** for all screen sizes  
  - Smooth **animations** for transitions and product loading  
  - **Shimmer effect** for better loading experience  

- **Payment**
  - Secure checkout using **Stripe**  

- **Architecture**
  - **MVVM** (Model-View-ViewModel) pattern for clean separation of logic  
  - State management with **Cubit (Bloc)**  
  - **Provider** for cart & wishlist management  
  - API integration using **Dio**  
  - Clean **Repository Pattern** for organized code
